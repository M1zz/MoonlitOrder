> 보호 대상입니다. (이 앱의 모든 게임 명칭·역할·문구는 오리지널로 작성됨)

전체 스타일 가이드
- 톤: 다크 네이비 밤하늘 배경(#141633 계열) 위에 금색(골드)·달빛 파랑 포인트. 앱 전체가 다크 테마.
- 무드: "달빛 아래 비밀결사" — 신비롭고 고전 판타지풍. 달, 별, 망토, 가면, 단검 모티프.
- 포맷: 배경 투명 PNG(@2x, @3x) 또는 벡터 PDF 1개(Xcode에서 Preserve Vector Data). 어두운 배경 위에 올라가므로 밝은 외곽 라인/발광 표현이 잘 어울림.
- 이름은 아래 `에셋 이름` 그대로 Assets.xcassets에 넣으면 코드 연결은 바로 가능.

## 참고

- 역할 카드 4종(1~4번)은 등장 빈도와 감정 몰입도가 가장 높아 최우선.
- 같은 화풍/선 굵기/팔레트로 통일해야 시스템 아이콘과 섞여도 이질감이 없음.
- 파일을 주시면 Assets.xcassets 등록과 코드 교체는 요청 시 바로 진행 가능.
