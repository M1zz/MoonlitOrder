# 달빛 오락실 — 일러스트 요청 문서 (전체 게임)

> ⚠️ **저작권 주의**: 모든 일러스트는 직접 그리거나 상업적 사용이 허용된
> 라이선스(구매·CC0 등)의 원본이어야 합니다. 기존 보드게임(아발론, 한밤의
> 늑대인간 등)·영화·드라마의 아트워크를 따라 그리거나 차용하면 안 됩니다.
> 게임 규칙은 저작권 보호 대상이 아니지만, 그림·명칭·문구 같은 '표현'은
> 보호 대상입니다. (이 앱의 모든 게임 명칭·역할·문구는 오리지널로 작성됨)

## 공통 스타일 가이드

- 톤: 다크 네이비 밤하늘 배경(#141633 계열) 위에 금색(#F2C759)·달빛 파랑(#73BFFF)·그림자 빨강(#F25A6B) 포인트. 앱 전체가 다크 테마.
- 무드: "달빛 아래 밤의 오락실" — 신비로운 판타지 + 한국 설화가 섞인 아늑한 밤 분위기.
- 포맷: 배경 투명 PNG(@2x, @3x) 또는 벡터 PDF 1개(Xcode에서 Preserve Vector Data). 어두운 배경 위에 올라가므로 밝은 외곽 라인·발광 표현이 잘 어울림.
- **세 게임의 화풍·선 굵기·팔레트를 통일**할 것 — 한 앱 안에서 섞여 보임.
- 이름은 아래 `에셋 이름` 그대로 Assets.xcassets에 넣으면 코드 연결 바로 가능.

## 제작 순서 제안 (총 35종)

1. **★★★ (14종)**: 게임 대표 이미지 3종(B1·C1·D1) → 달빛 결사 역할 카드 4종(B2~B5) → 달 없는 밤 역할 카드 5종(D2~D6) → 라이어 제시어/배정(C2·C3) → 앱 아이콘 교체(A1, 임시본 있음) + 홈 키 비주얼(A2)
2. **★★ (15종)**: 승리 화면 6종 → 미션/카드류 → 검거·밤/낮 장면 → 투표 인장
3. **★ (6종)**: 배경 텍스처, 발언·암살 장면 등 분위기 보강

- 역할 카드(3:4)는 등장 빈도와 몰입도가 가장 높아 최우선입니다.
- 파일을 주시면 Assets.xcassets 등록과 코드 교체는 요청 시 바로 진행합니다.
- 일부만 먼저 주셔도 됩니다 — 없는 에셋은 지금처럼 시스템 아이콘으로 표시됩니다.
